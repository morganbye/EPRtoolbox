PS_Plot1

PURPOSE

PS_PLOT1 Plot the raw power saturation spectra

SYNOPSIS

function PS_Plot1(handles)

DESCRIPTION 

 PS_PLOT1 Plot the raw power saturation spectra

 Syntax:  PS_PLOT1

 Inputs:
    input1 - handles
               define the GUI area (ie gcf for PowerSat)

 Outputs:
    output1 - Datasets plotted in PowerSat GUI

 Example: 
    see http://morganbye.net/PowerSat

 Other m-files required:
    PowerSat.m

 Subfunctions:         PS_PLOT1_PLOTALL
                       PS_PLOT1_PLOTSINGLE
                       PS_PLOT1_VARIABLES

 MAT-files required:   none

CROSS-REFERENCE INFORMATION

This function calls:
  • PS_Plot1_PlotAll PS_Plot1_PlotAll Plot all spectra from one experiment at once
  • PS_Plot1_PlotSingle PS_Plot1_PlotSingle Plot one spectrum from an experiment.
  • PS_Plot1_Variables PS_Plot1_Variables Open power saturation experiments
