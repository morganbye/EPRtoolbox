PURPOSE

PELDORPLOTTER take a set of PELDOR experiments and plot them quickly

DESCRIPTION 

 PELDORPLOTTER take a set of PELDOR experiments and plot them quickly

 PELDORPLOTTER ()

 PELDORPLOTTER files must be saved with the file format:
