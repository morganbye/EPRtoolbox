PURPOSE

SYNOPSIS

DESCRIPTION 

 AACONVERTER 
      AACONVERTER, is a simple tool for the obtaining of amino acid
      sequences in either 1 letter or 3 letter code from the other.

      Copy and Paste buttons are included for easy manipulation from and
      to the clipboard.
