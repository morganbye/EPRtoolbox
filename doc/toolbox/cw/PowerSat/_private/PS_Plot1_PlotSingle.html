PS_Plot1_PlotSingle

PURPOSE

PS_Plot1_PlotSingle Plot one spectrum from an experiment.

SYNOPSIS

function [mwPower] = PS_Plot1_PlotSingle(handles,graph,exp,color)

DESCRIPTION 

 PS_Plot1_PlotSingle Plot one spectrum from an experiment.

 Syntax:  PS_Plot1_PlotSingle

 Inputs:
    input1 - handles

    input2 - graph
               define the graph area (ie gca or handles.plot_load)
    input3 - exp
               experiment variable (ie Oxy,N2,Ni)
    input4 - color
               color of lines drawn on graph for peaks

 Outputs:
    output1 - mwPower
               the microwave power of the currently displayed spectrum

 Example: 
    see http://morganbye.net/PowerSat

 Other m-files required:
    PowerSat.m

 Subfunctions:         none

 MAT-files required:   none

CROSS-REFERENCE INFORMATION

This function calls:
  • PS_Plot1_Gaussian PS_Plot1_Gaussian Use a Gaussian fitting model to fit the model high and
  • PS_Plot1_GaussianDeriv PS_PLOT1_GAUSSIAN Use a Gaussian fitting model to fit the model high
  • PS_Plot1_Lorentzian PS_PLOT1_LORENTZIAN Use a Lorentzian fitting model to fit the model high
  • PS_Plot1_LorentzianDeriv PS_PLOT1_LORENTZIAN Use a Lorentzian fitting model to fit the model high
This function is called by:
  • PS_Plot1 PS_PLOT1 Plot the raw power saturation spectra
