PS_Pi

PURPOSE

PS_PI Calculate the Pi (accessibility) factor

SYNOPSIS

function PS_Pi(handles)

DESCRIPTION 

 PS_PI Calculate the Pi (accessibility) factor

 Syntax:  PS_PI(handles)

 Inputs:
    input1 - handles
               define the GUI area (ie gcf for PowerSat)

 Outputs:
    output1 - Power saturation curves plotted in PowerSat GUI

 Example: 
    see http://morganbye.net/PowerSat

 Other m-files required:
    PowerSat.m

 Subfunctions:         PS_PI_DPPH
                       PS_PI_SAMPLE
                       PS_PI_CALCULATE

 MAT-files required:   none

CROSS-REFERENCE INFORMATION

This function calls:
  • PS_Pi_Calculate PS_PI-CALCULATE Calculate the Pi (accessibility) factor
  • PS_Pi_DPPH PS_PI_DPPH Calculate ΔH_half for the DPPH standard
  • PS_Pi_Sample PS_PI_SAMPLE Plot the power saturation curves
This function is called by:
  • PS_PlotPS PS_PLOTPS Plot the power saturation curves
